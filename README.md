🚀 StartupDoctor

AI-powered multi-agent system that converts GitHub repositories into investor-grade startup analysis reports.

🧠 What it does

StartupDoctor analyzes any GitHub repository and automatically generates a structured startup evaluation including:

Startup summary
Problem being solved
Target users
Market opportunity
Competitive analysis
Risks & weaknesses
Monetization potential
Execution feasibility
Final investment decision
Success probability score
Founder + investor insights
Actionable roadmap
⚙️ How it works

StartupDoctor uses a multi-agent AI pipeline:

Understanding Agent
Reads GitHub repo + README
Extracts core idea and problem statement
Evaluation Agent
Scores startup across key metrics:
Market size
Competition
Monetization
Execution feasibility
Moat
Roadmap Agent
Generates execution strategy and next steps
Final Agent
Combines all outputs into a structured investor-style report
🧩 Tech Stack
Backend: Django
AI Engine: Groq API / LLM Agents
Agent System: Lemma SDK (multi-agent orchestration)
Frontend: HTML, CSS, JavaScript
Deployment: Render (ready)
Data Source: GitHub API

Evaluating startups is slow, subjective, and inconsistent.
